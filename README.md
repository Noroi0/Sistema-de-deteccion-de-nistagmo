# Prototipo de sistema de deteccion de nistagmo
Este repositorio contiene el desarrollo del prototipo para la detección automática de nistagmo
utilizando técnicas de visión por computadora e inteligencia artificial.

## Contenido
- Código del sistema
- Documentación técnica
- Manual de usuario
- Reporte final del proyecto

## Documentación
La documentación completa se encuentra en la carpeta `docs/`.

## Tecnologías
- Python
- OpenCV
- Scikit-learn
- Pandas
- Tensorflow
- Tkinter
- SQL
- HTML
- FastAPI
- Redes neuronales (CNN + LSTM)
